# Development Reports

Weekly development reports for stakeholder communication.

## Structure

```
docs/reports/
├── README.md       # This file
├── TEMPLATE.md     # Report template
└── YYYY-WNN.md     # Weekly reports (e.g., 2026-W01.md)
```

## Usage

1. Copy `TEMPLATE.md` to new file: `YYYY-WNN.md`
2. Fill in week's achievements
3. Update milestone progress
4. Review and send to stakeholders

## Naming Convention

- Format: `YYYY-WNN.md`
- Example: `2026-W01.md` (Year 2026, Week 1)
