
## 📄 Yorutsuke V4.8 "Hybrid Ledger" 设计文档

### 1. 整体架构 (Overall Architecture)

* **设计隐喻**：**“数字坐标纸上的财务仪表盘”**。利用 24px 的点阵网格（Grid）作为底层视觉锚点，所有的 UI 元素都像是贴在精密图纸上的透明标签。
* **空间布局**：左侧为 256px 宽度的深色控制区，右侧为自适应宽度的数据画布区。
* **深度感 (Depth)**：通过 `backdrop-filter: blur(12px)` 和 `white/85` 的磨砂材质，构建了从“网格层”到“内容层”的物理景深感。
* **排版规则**：金融数据采用 **JetBrains Mono**（等宽字体）以确保小数点垂直对齐，非数值文本采用 **Inter**（人文无衬线）以提升阅读舒适度。

---

### 2. 左侧菜单栏 (Sidebar System)

* **基座**：采用深度为 `#0F172A` 的午夜蓝，象征财务管理的严肃与稳重。
* **品牌识别**：Logo 区域采用蓝色圆角矩阵（`rounded-2xl`），与右侧的尖锐边缘形成对比。
* **导航态势**：
* **激活项**：使用 `Blue-600/10` 背景色与 `4px` 垂直强调色条。
* **动态反馈**：所有条目均具备 `all 0.25s` 的平滑过渡，Hover 时从暗灰转向亮白，提升交互的仪式感。


* **用户组件**：底部固定的 User Widget 采用半透明边框，将个人账户信息作为系统的“末端”进行展示。

---

### 3. 主显示区 (Main Display)

主显示区被划分为三个逻辑模块，按“过滤 -> 统计 -> 明细”的顺序自上而下排列。

#### A. 时间控制器 (Temporal Controller)

* **双行并行逻辑**：
* **Yearly 行**：作为顶层过滤器。当前年份（2025）下方配有**蓝色呼吸点（Active Dot）**，并带有微妙的光晕阴影。
* **Interval 行**：提供**“Full Year”**全局视图与**“Specific Months”**局部视图的切换。利用虚线或 1px 垂直细线进行逻辑分割。


* **交互逻辑**：采用“零阻力”设计，所有时间单位均为点击即响应，无需确认。

#### B. 数据看板 (Summary Tiles)

* **卡片视觉**：采用双列布局。
* **Revenue 卡片**：Emerald-400（祖母绿）立边 + 渐变底色，强调资金流入。
* **Expense 卡片**：Rose-400（玫瑰红）立边 + 渐变底色，预警资金流出。


* **数据呈现**：大号加粗的等宽字体数值（¥），赋予财务数据极强的视觉冲击力。

#### C. 原子化卡片列表 (Universal Item Stream)

* **卡片结构**：
1. **Date Block (左)**：采用 `Slate-100` 背景的方形块，清晰显示月份与日期，方便用户按时间轴快速扫视。
2. **Detail Block (中)**：主标题显示交易对手方，副标题使用 **微型标签 (Tags)** 展示分类（如 SUBSCRIPTION）和账本备注。
3. **Value Block (右)**：金额采用正负号区分。收入为 Emerald 色，支出为 Rose 色，底部标注具体的支付账户（Account Label）。


* **悬浮特效**：卡片在 Hover 时会产生 `translateY(-2px)` 的升起效果和阴影加深，赋予静态列表以动态生命感。
