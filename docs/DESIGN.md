## Component Library

### Buttons

| Variant | Usage |
|---------|-------|
| Primary | Main actions (Confirm, Save) |
| Secondary | Cancel, alternative actions |
| Danger | Delete, destructive actions |
| Ghost | Subtle actions (Edit, More) |

### Forms

- Input fields with labels above
- Validation messages below in red
- Required fields marked with *

### Cards

- Rounded corners (8px)
- Subtle shadow in light mode
- Distinct background in dark mode

- All interactive elements focusable
- Keyboard navigation support
- Screen reader labels
